# Development

## Getting Started

First, run the development server:

```bash
bun run dev
```

To run the build:

```
bun run vercel:build
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployments

The website will be deployed automatically on a push to `master`
